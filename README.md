# Zameer Cabs - Vehicle Rental Booking App

A full-stack vehicle rental booking application for Zameer Cabs, built with React, Node.js, Express, and MongoDB.

## Features
- User Authentication (JWT)
- Vehicle Browsing & Search
- Booking Management
- Admin Dashboard
- Responsive Design

## Getting Started

### Backend
1. `cd backend`
2. `npm install`
3. Create a `.env` file with your MongoDB URI and JWT Secret.
4. `npm run dev`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`
